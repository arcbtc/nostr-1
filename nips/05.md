NIP-05
======

Follow List
-----------

`draft` `optional`

All the keys a public key is following

## `kind` `055` `followers`

`05`: `followers`: has no `content` 

`e` `tag` for each public key following

The relay should keep the public key's most recent  `kind` `05` event, and discard the rest
